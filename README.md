## What Is This, Exactly?

If you've never heard of Emby or Plex — they're media server platforms. You point them at a library of movies, TV shows, music, whatever, and they organize everything with nice artwork, metadata, and a clean interface. Then you can stream from any device: phone, TV, browser, you name it.

The traditional route is self-hosting: you run the server software on a computer at home, manage storage, keep things updated. MediaRepo flips that — they host the server, you just get a user account. Think of it like a private Netflix, but someone else handles the technical headache.

MediaRepo offers all three major platforms: Emby, Plex, and Jellyfin. Each has a slightly different personality, and MediaRepo is upfront about which one might suit you better.

## Emby vs Plex: Which Should You Pick?

This comes up constantly in the media server community, and MediaRepo actually gives a useful hint on their own site. Here's a quick take:

**Go with Emby if:** You want all apps to just work without needing an extra subscription. Emby apps being automatically unlocked is a real convenience win, especially on smart TVs and streaming devices.

**Go with Plex if:** You're an anime watcher and subtitle rendering matters to you. Plex's ASS/SSA transcoding is notably better, and if you're deep into fansub territory, this is a practical difference, not a theoretical one.

**Go with Jellyfin if:** You philosophically prefer open-source software and the $7/mo price point fits better. Jellyfin has no licensing headaches — it's just pure community-built software.
